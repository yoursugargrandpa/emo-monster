# 本地啟動指南

## 快速開始

### 1. 安裝依賴
```bash
npm install
```

### 2. 啟動本地開發伺服器
```bash
npm run dev
```

伺服器將在以下位址啟動：
- **本地**：http://localhost:5173/
- **網路**：http://10.0.11.21:5173/

### 3. 建立生產版本
```bash
npm run build
```
輸出將在 `dist/` 目錄

### 4. 預覽生產版本
```bash
npm run preview
```

### 5. 運行測試
```bash
npm test
```

## 功能說明

### 🎨 情緒色彩混合
- 拖放表情圖片到畫布進行色彩混合
- 支援加法混合模式（疊加顏色更亮）
- 即時顯示合成顏色代碼

### 🥚 情緒蛋孵化
- 點擊「獲得情緒蛋（模擬）」獲得隨機顏色的蛋
- 點擊「孵化情緒蛋」孵化單個蛋
- 點擊「孵化全部」同時孵化所有蛋
- 孵化時有粒子特效和音效反饋

### 📈 怪獸進化
- 孵化後的怪獸自動升級（達到經驗值門檻）
- 點擊「進化最近孵化的怪獸」手動確認進化
- 進化 Modal 預覽升級效果
- 已裝備配件進化時自動保留

### 🛍️ 商店系統
- 點擊「進入商店」開啟商店介面
- 購買配件（共 6 種：翅膀、角、皇冠、星星、火焰、心形）
- 完成挑戰獲得 10 金幣
- 查看背包中已購買的配件

### 🎖️ 配件系統
- 為怪獸裝備購買的配件
- 在左側「怪獸配件」區域查看已裝備配件
- 進化時配件自動保留

### 📅 情緒時光機
- 查看所有孵化的怪獸歷史
- 按日期瀏覽當日孵化的怪獸
- 週期高亮（7 天間隔）

### ⚙️ 設定與匯出
- **粒子數量**：調整粒子特效密度
- **粒子大小**：調整粒子大小
- **音量**：調整音效音量
- **進化門檻基數**：調整升級所需經驗值（預設 5）
- **匯出格式**：
  - PNG（圖片）
  - JSON（資料）
  - SVG（向量圖）
  - Bundle（含 metadata 的 JSON）

## 數據儲存

所有數據都儲存在瀏覽器的 `localStorage` 中：
- `emo_eggs`：情緒蛋資料
- `emo_monsters`：怪獸資料（含配件）
- `emo_coins`：金幣餘額
- `emo_inventory`：背包物品
- `emo_settings`：使用者設定

## 瀏覽器相容性

支援所有現代瀏覽器：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 技術棧

- **React 18** - UI 框架

## 常見問題

### Q: 開發伺服器無法啟動？
A: 確保 Node.js 已安裝 (`node -v`) 且已執行 `npm install`

### Q: 匯出的檔案在哪？
A: 匯出的檔案會自動下載到瀏覽器的預設下載目錄

### Q: 如何清除所有數據？
A: 在瀏覽器開發者工具 (F12) → Application → Local Storage 中刪除相關項目

## 開發提示

- 使用 `F12` 開啟瀏覽器開發者工具進行除錯
- Console 會顯示所有錯誤和警告信息
- Network 標籤可查看所有網路請求
- 修改代碼時會自動熱更新（Hot Reload）
