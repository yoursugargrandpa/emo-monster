# 🎨 情感怪獸 - 設計系統

## 設計願景

情感怪獸是一個將**情感具象化**的互動應用。通過色彩混合、怪獸孵化、進化等機制，讓用戶以創意和娛樂的方式探索和表達自己的情感。

## 色彩系統

### 主色調
- **Primary 紅色**：`#ff6b6b` - 活力、熱情、行動
- **Secondary 青色**：`#4ecdc4` - 平靜、信任、精準
- **Accent 金色**：`#ffd93d` - 光明、獎勵、成就

### 情緒色
- **生氣**：`#ff6b6b` (紅色)
- **難過**：`#6bcaff` (藍色)
- **快樂**：`#ffd93d` (金色)

### 中性色板
- **背景淺色**：`#f8f9fa` - 柔和背景
- **背景深色**：`#2d3436` - 高對比背景
- **文字淺色**：`#ffffff` - 淺色文字
- **文字深色**：`#2d3436` - 深色文字
- **邊框**：`#e0e0e0` - 細邊框線

## 排版系統

### 字體族
```css
-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen', 'Ubuntu', 'Cantarell', sans-serif
```

### 文字大小與權重
- **標題 (H1)**：32px, 700 (Bold)
- **標題 (H2/H3)**：16px-24px, 600 (SemiBold)
- **正文**：14px, 400 (Regular)
- **說明文字**：12px-13px, 500 (Medium)
- **標籤**：12px, 600 (SemiBold)

## 間距系統

| 變數 | 值 | 用途 |
|------|-----|------|
| `--spacing-xs` | 4px | 最小元素間距 |
| `--spacing-sm` | 8px | 表單項目間距 |
| `--spacing-md` | 12px | 卡片內間距 |
| `--spacing-lg` | 16px | 主要容器間距 |
| `--spacing-xl` | 24px | Modal 內間距 |
| `--spacing-2xl` | 32px | 大型分隔符 |

## 圓角系統

| 變數 | 值 | 用途 |
|------|-----|------|
| `--radius-sm` | 4px | 表單元素 |
| `--radius-md` | 8px | 卡片、按鈕 |
| `--radius-lg` | 12px | 主要容器 |
| `--radius-xl` | 16px | 大型模態框 |

## 陰影系統

### 層級陰影
- **Small**：`0 2px 8px rgba(0, 0, 0, 0.08)` - 微妙強調
- **Medium**：`0 4px 16px rgba(0, 0, 0, 0.12)` - 標準深度
- **Large**：`0 8px 24px rgba(0, 0, 0, 0.15)` - 頂級深度

## 動畫與過渡

### 過渡速度
- **Fast**：150ms - 快速互動反饋
- **Normal**：300ms - 標準過渡
- **Slow**：500ms - 強調動畫

### 常用緩動函數
```css
cubic-bezier(0.4, 0, 0.2, 1)
```
流暢、自然的動畫曲線

## 互動設計

### 按鈕狀態
- **正常**：陰影 sm，完整色彩
- **Hover**：上移 2px，陰影 md，色彩加深
- **Active**：返回原位，陰影 sm
- **Disabled**：灰色，不透明度 60%，禁用游標

### 表單元素
- **焦點**：邊框色變為主色
- **錯誤**：邊框色變為紅色
- **成功**：邊框色變為綠色

## 響應式斷點

### 桌面版
- **最小寬度**：1280px
- **佈局**：2 列（畫布 + 控制面板）

### 平板版
- **寬度範圍**：768px - 1024px
- **佈局**：2 列（寬度調整）

### 手機版
- **最大寬度**：767px
- **佈局**：1 列（上下排列）

## 元件設計

### Card（卡片）
```
外邊距: 12px
內邊距: 12px
背景: 白色
邊框: 1px 淺灰色
圓角: 8px
陰影: Small
```

### Button（按鈕）
```
內邊距: 8px 12px
背景: 主色或副色
文字色: 白色
圓角: 8px
邊框: 無
陰影: Small
過渡: 150ms
```

### Input（輸入框）
```
內邊距: 8px
邊框: 1px 淺灰色
圓角: 4px
背景: 白色
焦點邊框: 主色
```

### Modal（模態框）
```
最大寬度: 500px
最大高度: 80vh
內邊距: 24px
背景: 白色
圓角: 12px
陰影: Large
背景覆蓋: rgba(0,0,0,0.5) + 模糊 4px
```

## 品牌視覺元素

### Logo 設計
- 使用 Emoji「🎨」作為簡易 Favicon
- 可擴展為自訂圖形 Logo

### 漸層背景
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
```
紫色到紫羅蘭色的優雅漸層

### 動畫序列
- **怪獸孵化**：粒子爆發 + 縮放動畫
- **進化過程**：脈動 + 粒子雨
- **UI 加載**：淡入 + 向上滑入

## 可訪問性（Accessibility）

### 色彩對比
- **AA 標準**：4.5:1 以上（文字）
- **AAA 標準**：7:1 以上（重要文字）

### 焦點指示器
- 所有交互元素都有清晰的焦點狀態
- 使用鍵盤可完整導航應用

### 動畫安全
- 提供 `prefers-reduced-motion` 支援
- 關鍵動畫可被禁用

## 設計令牌實現

所有設計令牌都定義在 CSS 變數中，便於維護和切換主題：

```css
:root {
  --primary: #ff6b6b;
  --secondary: #4ecdc4;
  /* ... 更多令牌 */
}
```
